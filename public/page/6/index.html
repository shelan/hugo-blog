How to change default temporary folder in WSO2 carbon based products

Mar 11, 2012 · 1 minute read · Comments In wso2 carbon based products default temporary folder is {product_home}/tmp.  If you need to change that location to a different location here is the way to do it.  You can change the location by changing the locations in start up scripts.  1) In Linux (/bin/wso2server.sh) change following parameter’s location.  -Djava.io.tmpdir=“$CARBON_HOME/tmp”  2) In Windows (/bin/wso2server.bat) change the following parameter’s location.  -Djava.io.tmpdir=“%CARBON_HOME%\tmp”
